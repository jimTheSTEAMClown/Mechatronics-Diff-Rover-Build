# Mechatronics-Diff-Rover-Build
Big & Small Diff Rovers built for my Mechatronics Engineering Class

## Welcome
The plan is to both document my own build of a Differential Drive Rover, where it has an Arduino & a Raspberry Pi. The goal will be to create both a set of step by step Labs, but also a story and build process.  So the top of this doc will be Labs, and lower down will be my blog of the creative process to build this Rover.  The plan will be to document building a Differential Drive Rover, firstly controlling it with an Arduino (probably an Arduino Mege, and then adding a Raspberry Pi.  The order, at first might be the Pi, as I'm starting this mid semester with my students... but eventually it will be 1) Build Rover Frame, 2) Driver With Arduino, 3) Upgrade To Have Raspberry Pi Driver It
